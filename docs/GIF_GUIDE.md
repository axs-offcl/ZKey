# GIF Recording Guide

Record your GIFs using [ScreenToGif](https://www.screentogif.com/) (free, Windows) for professional results.

## Recommended Settings

- **FPS**: 60
- **Resolution**: Record at 1x (no scaling)
- **Duration**: 5-10 seconds max per GIF
- **Cursor**: Enable cursor recording

## GIFs to Record

| Filename | Content |
|----------|---------|
| `overlay_demo.gif` | Show overlay in action — press keys, highlight them, show mouse clicks |
| `settings_demo.gif` | Open settings (Ctrl+Shift+K), change theme, adjust opacity |
| `plugin_demo.gif` | Install a plugin from Marketplace, enable it, show config UI |
| `editor_demo.gif` | Open Layout Editor, drag a key, resize it, change colors |

## Tips for Smooth GIFs

1. **Move mouse slowly** — 2-3 seconds per movement
2. **Pause before actions** — let viewers see what you're about to click
3. **Keep it short** — 5-10 seconds per GIF
4. **Crop tightly** — remove unnecessary desktop area

## Post-Processing

1. Open at [ezgif.com](https://ezgif.com/optimize)
2. Upload your GIF
3. **Optimize** — reduce to <2MB for GitHub
4. **Resize** — max width 800px works well in READMEs
5. Download and place in `docs/`
